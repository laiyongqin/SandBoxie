2D画面いける座標点にテレポートする際に発生していた
画面が一瞬、黒くなる現象を遅延時間を増やすことで解決した。
|
|
ThirdPersonCaractorブループリント→ビューポート
コンポーネント内にある、CameraBoom→詳細タブ→Lag
→Enable CameraLagにチェック
Camera Lag Speedを5.0に変更。
↑
処理内容
	Lagを有効化することで、カメラが追従する際に少し遅れて追従することを許可。
	Camera Lag Speedの値が小さいほど遅延時間が増える。

2D画面における黒画面現象を新たの方法で解決した。
レベル上に配置した PlayerStartアクター→詳細タブにある
CapsuleComponent→Shape→Capsule Half Height の値を見る。
テレポート先の高さ座標値 + Capsule Half Height + ( Capsule Half Height
 ÷ 4 ) の座標値をテレポート先の座標値の高さに入れる事で解決した。

この事象は, SetActorLocationした時に、
一瞬キャラが地面に埋まっているかもしれません。
レベルエディタでPlayer Start アクターを配置しておいて、
そのPlayer Start アクターの座標にキャラをSetActorLocationすれば、
地面に埋まりません。

場所によって計算式に不確実性があるので, とりあえず黒画面が発生するときは
テレポート先の座標にレベル上から, PlayerStartアクターを置いて
その PlayerStartアクター座標を SetActorLocation に入れてみる。

従って, 前回, 変更を加えた Enable Camera Lag は, チェックを外し
無効化した、カメラの遅延は発生しない。
